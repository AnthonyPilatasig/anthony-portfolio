# 🚀 Anthony Pilatasig - Professional Portfolio

Bienvenido al código fuente de mi portafolio personal. Soy un **Desarrollador Full Stack** con más de 2 años de experiencia profesional construyendo y escalando aplicaciones web, actualmente cursando la carrera de Ingeniería de Software.

Este repositorio contiene la landing page interactiva que utilizo para presentar mis proyectos, mi experiencia dual y mis habilidades técnicas.

## 🛠️ Tecnologías Principales (Tech Stack)

El portafolio está construido siguiendo estándares modernos de alto rendimiento y diseño:

*   **React 19 & TypeScript:** Tipado estricto y la última versión de React.
*   **Vite:** Herramienta de compilación ultrarrápida.
*   **Tailwind CSS v4:** Motor de utilidades CSS para un diseño ágil, responsivo y mantenible.
*   **Framer Motion:** Librería para micro-animaciones, efectos de *Glassmorphism* fluidos y transiciones interactivas complejas.
*   **React Icons:** Optimización y carga rápida de iconos vectoriales.

## 💡 Características Clave de la Arquitectura
- **Mobile First & Fully Responsive:** Diseñado para funcionar a la perfección en cualquier tamaño de pantalla.
- **Glassmorphism Design System:** Efectos visuales de cristal y degradados de neón personalizados mediante variables CSS puras.
- **Optimización de Renderizado:** Componentes atomizados y animaciones por hardware.

## 🚀 Instalación Local

Si deseas correr este proyecto en tu entorno local:

1. Clona el repositorio:
   ```bash
   git clone https://github.com/AnthonyPilatasig/anthony-portfolio.git
   ```
2. Instala las dependencias:
   ```bash
   npm install
   ```
3. Inicia el servidor de desarrollo local:
   ```bash
   npm run dev
   ```
4. Abre `http://localhost:5173` en tu navegador.
